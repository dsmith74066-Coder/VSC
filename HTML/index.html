Important Tips

  • Nesting: Tags must be properly nested. If you open a tag inside another, close it before closing the parent.
  • Case:HTML is not case-senitive, but lowercase is standard.
  • White scpace:Extra space and line breaks in your code don't affect display.
  • Comments:use <!-- comment here --> to leave notes in your code.

The DOM Tree

HTML create a tree struture called the Document Object Model (DOM):

html (root)
├── head
│   ├── meta
│   └── title
└── body
    ├── h1
    ├── p
    │   └── strong
    └── div
        ├── h2
        └── p

Each element is a "node" in the tree, with parent-child relationships.

style="margin-top: 40px; padding-top: 20px; border-top: 1px; solid #ccc; text-align: center; color: #666">
